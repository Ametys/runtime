Tests of the client side of the Ametys UI

List of tests

  • User interface: Test of the purely drawn client side (no logic). Allow to test the rendering of the ribbon and of the tools.
  • Tools: Test of the logic of the tools. Allow to test the abstraction of the tools.

How to make tests working

To make the test working, you have to play thoses throught Apache HTTPD and to add some url redirection:

  • /plugins/extjs6/resources/: Redirection to the subdirectory "build" of your local copy of ExtJS version
  • /kernel/resources/: Redirection to the kernel resources directory

Here is a Apache Httpd 2.2 conf sample:

				<VirtualHost *:80>
				        # Racine des documents statiques du site web
				        DocumentRoot "/pathto/Ametys/kernel/01_RUNTIME/trunk/test/clientside"
				        
	                        <Directory "/pathto/Ametys/kernel/01_RUNTIME/trunk/test/clientside">
	                           AllowOverride none
	                           Allow From All
	                           Options MultiViews  
	                        </Directory> 
				            <Directory "/pathto/Ametys/plugins/core-ui/resources">
				               AllowOverride none
				               Allow From All
				               Options MultiViews  
				            </Directory> 
				            <Directory "/pathto/Extjs/ext-6.0.0/build">
				               AllowOverride none
				               Allow From All
				               Options MultiViews  
				            </Directory> 
				            
				        # Nom du virtual host
				        ServerName ribbon.localhost
				        UseCanonicalName On
				        RewriteEngine On
				
				        AddDefaultCharset     UTF-8
				        
				        RewriteRule             ^/?$             /index.html [R]
				        RewriteRule             ^/plugins/extjs6/resources/(.*)$     /pathto/Extjs/ext-6.0.0/build/$1 [L]  
				        RewriteRule             ^/plugins/core-ui/resources/(.*)$    /pathto/Ametys/plugins/core-ui/resources/$1 [L]
				</VirtualHost>
